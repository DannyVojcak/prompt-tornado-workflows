# SaaS Analytics Platform Design Workflow

This workflow demonstrates how Prompt Tornado can generate structured system architecture designs from a single prompt.

The workflow analyzes a technical problem, identifies core system components, and produces a structured architecture explanation.

## What this workflow demonstrates

- structured technical reasoning
- system architecture synthesis
- technology stack recommendations
- scalable infrastructure design

## Example Steps

1. Task classification  
2. Architecture reasoning  
3. Component identification  
4. Technology stack mapping  
5. Structured output generation

## Example Pipeline

Prompt  
↓  
Task Classification  
↓  
Architecture Analysis Agent  
↓  
Technology Stack Recommendation Agent  
↓  
Structured Output
